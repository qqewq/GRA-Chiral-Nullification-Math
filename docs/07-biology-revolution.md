# Революция в биологии

## Гомохиральность как следствие GRA-обнулёнки в пребиотической эволюции

### Модель

Пребиотическая супь содержала рацемическую смесь аминокислот:

$$\text{Политика}_0 = (0.5, 0.5), \quad W_0 = \{L: 0.5, D: 0.5\}$$

Применение GRA-обнулёнки:

$$N(\text{Политика}_0, W_0, S_{\text{prebiotic}}) = ((R, L), 0, 0)$$

### Эволюционный механизм

1. **Фаза 1:** Саморепликация с ошибками создаёт флуктуации.
2. **Фаза 2:** GRA-обнулёнка подавляет симметричные стратегии.
3. **Фаза 3:** Остаётся чистая хиральность (L-форма).

### Предсказание

Теория предсказывает, что в любой самореплицирующейся системе с зеркальной симметрией неизбежно возникнет гомохиральность — это не случайность, а математическая необходимость.
