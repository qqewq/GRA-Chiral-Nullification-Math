# Математика GRA-обнулёнки

## Формальное определение

GRA-обнулёнка — это оператор:

$$N: (\Pi, W, S) \mapsto (\Pi', 0, 0)$$

где $\Pi' = (R, L)$ — хиральная пара, $W$ и $S$ зануляются.

## Свойства

1. **Идемпотентность:** $N(N(x)) = N(x)$
2. **Хиральность образа:** $\text{Im}(N) \subset H(X)$, где $H(X)$ — пространство хиральных состояний.
3. **Универсальность:** Применима к любому объекту с симметрией.
