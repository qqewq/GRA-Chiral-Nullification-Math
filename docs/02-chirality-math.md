# Математика хиральности

## Определения

**Определение 1 (Зеркальный оператор).** Пусть $X$ — метрическое пространство. Оператор $Z: X \to X$ называется зеркальным, если $Z^2 = \text{id}$.

**Определение 2 (Хиральность).** Объект $x \in X$ хирален относительно $Z$ и группы вращений $SO$, если:
$$\forall \rho \in SO: \rho(x) \neq Z(x)$$

## Теорема существования

Для любого объекта с нетривиальной структурой существует оператор $Z$, относительно которого объект хирален.
