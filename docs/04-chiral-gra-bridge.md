# Хиральный GRA-мост

## Теорема эквивалентности

**Теорема.** Пусть $Z$ — инволютивный оператор зеркального отражения ($Z^2 = \text{id}$). Тогда для любого объекта $X$ и группы вращений $SO$ такого, что $\forall\rho\in SO: \rho(X) \neq Z(X)$, существует изоморфизм:

$$H(X) \cong N_{\Pi}(X, W, S) = (X, Z(X), 0, 0)$$

## Доказательство

### Шаг 1: Построение отображения

Определим $\Phi: H(X) \to \text{Im}(N)$ по правилу:
$$\Phi(x) = (x, Z(x), 0, 0)$$

### Шаг 2: Инъективность

Пусть $\Phi(x_1) = \Phi(x_2)$. Тогда $x_1 = x_2$ по определению первой компоненты.

### Шаг 3: Сюръективность

Любой элемент образа $N$ имеет вид $(x, Z(x), 0, 0)$, что и есть $\Phi(x)$.

### Шаг 4: Сохранение структуры

Метрика на $H(X)$ индуцирует метрику на образе $N$ через первую компоненту.

## Следствия

1. Любое обнуление симметрии порождает хиральность.
2. Хиральность — необходимое следствие оператора $N$.
3. Все научные дисциплины, изучающие асимметрию, описываются единым формализмом.
