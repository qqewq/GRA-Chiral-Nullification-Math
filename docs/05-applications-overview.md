# Обзор приложений

GRA-хиральность применима ко всем наукам, изучающим асимметрию:

| Наука | Асимметрия | Оператор $Z$ |
|-------|-----------|--------------|
| Физика | CP-нарушение | Комплексное сопряжение |
| Биология | Гомохиральность | L/D-отражение |
| Химия | Энантиоселективность | Инверсия конфигурации |
| Лингвистика | Актив/пассив | Синтаксическое зеркало |
| Экономика | Кризисы | Обмен спрос/предложение |
| Нейронаука | Латерализация | Сагиттальное отражение |
| Космология | Барионная асимметрия | Материя/антиматерия |
| Социология | Левые/правые | Полное отражение |
| Эстетика | Красота | Инверсия пропорции |
| Квантовые вычисления | Кубитные состояния | Комплексное сопряжение |
